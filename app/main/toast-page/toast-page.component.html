Floating Toast

Change the active value with a click event (use ViewChild on #Toast)

Display Toast I'm a toast ! 
              
                  <c3m-toast #toast styleClass="alert" position="bottom-right" closeBtn="true" closeBtnTitle="Close alert"> 
                    I'm a toast !
                </c3m-toast>

Properties

Name Type Default Description
active boolean null display the toast
delay number Not null delay before toast dissapears
styleClass string Null Style class of the component
position string top-right Toast position :
  • top-right
  • top-left
  • bottom-left
  • bottom-right
closeBtn boolean false Add a close button
imgCloseBtn string White 'X' URL for the close button img
closeBtnTitle string Close Label for the close button img
