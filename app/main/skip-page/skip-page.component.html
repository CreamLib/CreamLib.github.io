Require

Before using anchor, you have to add some configurations to your routing system.

In your app.route file, add :

export const routerOptions: ExtraOptions = {
  useHash: true,
  anchorScrolling: 'enabled'
};

In your app.module file, add :

RouterModule.forRoot(appRoutes, routerOptions),

Skip Link (invisible)

This link allows a screen-reader or keyboard user to skip big chunks of navigation links. The link is displayed only when it gets focus with keyboard.

Put focus on it to show it

Go to the main content 
<c3m-skip [target]="'goTotheBottom1'" isHidden="true">Go to the main content</c3m-skip>

I am the target of c3m-link

Skip Link

Go to the main content 
<c3m-skip [target]="'goTotheBottom2'">Go to the main content</c3m-skip>

I am the target of c3m-link

Properties

Name Type Default Description
target string Not null Id of targeted section
isHidden boolean false Set to true if you want to hide the link
styleClass string Null Class for the component
