Pagination

A simple page pagination.

  <c3m-pagination imgFirst="/assets/img/first.svg" imgBack="/assets/img/previous.svg" imgNext="/assets/img/next.svg"
  imgLast="/assets/img/last.svg"></c3m-pagination>

Properties

Name Type Default Description
pageSize number Null allows you to change the value of items per page (You will need to download pager.service.ts
imgFirst string Not null Image for the return to first page button
imgBack string Not null Image for the previous button
imgNext string Not null Image for the next button
imgLast string Not null Image for the go to last page button
