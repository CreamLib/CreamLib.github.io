Multiple Steps Github

This component displays the different steps of a form if divided in different pages/containers. It will automatically detect the previous and next steps. 

  <!-- STEP -->
  <c3m-step>
    <!-- STEP ITEMS -->
    <c3m-step-item isPassed="true" stepLink="#" title="Title Of Past Step"></c3m-step-item>
    <c3m-step-item isActive="true" title="Title Of Active Step"></c3m-step-item>
    <c3m-step-item title="Title Of Future Step"></c3m-step-item>
    <c3m-step-item title="Title Of Future Step"></c3m-step-item>
    <c3m-step-item title="Title Of Future Step"></c3m-step-item>
  </c3m-step>

Properties

Name Type Default Description
isPassed boolean null Define if a step is valid
stepLink string null Link to go back to a step (only is a step is passed)
isActive boolean null Define the active step
title string null Label of the step
