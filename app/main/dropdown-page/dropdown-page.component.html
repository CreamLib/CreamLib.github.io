Dropdown Component

  • Simple Action Dropdown

Dropdown

The Dropdown component can be combined with "c3m-button" or "c3m-button-link" components, or directly with a native link or a button.

Action 1 Action 2 Action 4 
<c3m-dropdown nameDropdown="Dropdown">
      <c3m-dropdown-item nameDdItem="Item1" url="creamng.github.io"></c3m-dropdown-item>
      <c3m-dropdown-item >
        <c3m-button>Action 1</c3m-button>
      </c3m-dropdown-item>
      <c3m-dropdown-item nameDdItem="Item3"></c3m-dropdown-item>
</c3m-dropdown>

Properties

Name Type Default Description
nameDropdown string Not null Label for the dropdown button
nameDdItem string Not null Label for the dropdown item (if using the c3m-dropdown-item component)
