Dialog Modal

Opens a dialog modal with focus set on the modal title. Focus loops inside the modal.

Properties

Name Type Default Description
modaTitle string Not null Title of the modal
styleClass string null Class for the component
topPosition number null Top coordinate of the modal
leftPosition number null Left coordinate of the modal. It will be interpretate in percentage (for responsive)
modalWidth number null Width of the modal. No useful if leftPosition is set.
closeBtnTitle string null Aria-label for close button

Events

Name Parameter Description
onShow event: Modal object Callback to invoke when dialog is shown.
onHide event: Modal object Callback to invoke when dialog is shown.
