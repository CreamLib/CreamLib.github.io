Radios

Rememeber that all radios from the same group must share the same name attribute value.

title
The label description
name
The value of the name attribute
value
The value of the input value attribute
id
The value of the id attribute
checked
if present, checkbox will be checked (optional)
required
if present, check will be required on submit (optional)
disabled
disable the checkbox (optional)
classes (optional)
go-along, switch

The id attribute of an input is very important for form accessibility. It allows the user-agent to associate a label to its input.

Single line radios (add the go-along class)

Switch radios (add the switch class)

<!-- RADIOS -->
<c3m-radio legend="Which star do you prefer?">
  <c3m-radio-item
    title="Star Wars"
    name="series"
    value="series1"
    [id]="'series1'">
  </c3m-radio-item>
  <c3m-radio-item
    title="Star Trek"
    name="series"
    value="series2"
    [id]="'series2'">
  </c3m-radio-item>
  <c3m-radio-item
    title="Star Gate"
    name="series"
    value="series3"
    [id]="'series3'">
  </c3m-radio-item>
</c3m-radio>

<!-- RADIOS IN-LINE -->
<c3m-radio class="go-along" legend="Star Trek is the best?">
  <c3m-radio-item
    title="Yes"
    name="startrekisbest"
    value="startrek"
    [id]="'best1'">
  </c3m-radio-item>
  <c3m-radio-item
    title="No"
    name="startrekisbest"
    value="other"
    [id]="'best2'">
  </c3m-radio-item>
</c3m-radio>
