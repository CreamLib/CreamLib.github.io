Checkboxes

title
The label tag
id
The most important attribute : Link between Label and Input
name
The name of input
class
allows to customize checkboxes (optional)
checked
if present, checkbox will be checked (optional)
required
if present, check will be required on submit (optional)
disabled
disable the checkbox (optional)

Custom checkbox (add the custom class)

Unstressed labels (add the unstressed class)

On-Off Button (add the on-off class)

<!-- SIMPLE CHECKBOX -->
<c3m-checkbox
  title="Do you want check me?"
  name="checkboxName1"
  value="checkboxValue1"
  [id]="'checkbox1'">
</c3m-checkbox>


<!-- CUSTOM CSS CHECKBOX -->
<c3m-checkbox
  title="Do you want check me?"
  name="checkboxName2"
  value="checkboxValue2"
  [id]="'checkbox2'"
  class="custom">
</c3m-checkbox>

<!-- UNSTRESSED LABELS -->
<c3m-checkbox
  title="Do you want check me?"
  name="checkboxName3"
  value="checkboxValue3"
  [id]="'checkbox3'"
  class="unstressed">
</c3m-checkbox>
