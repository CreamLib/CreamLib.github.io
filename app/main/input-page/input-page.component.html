Regular Text Input

Enter your name
id
input's identifier
name
input's general identifier
value
input's current value (optional)
autocomplete
defaults to "on" (optional)
maxlength
the maximum allowed value length (optional)
required
when present, the element is required (optional)
disabled
when present, the element is disabled/read-only (optional)
placeholder
Placeholder value (optional)

The id attribute of an input is very important for form accessibility. It allows the user-agent to associate a label to its input.

Placeholders don't replace labels; they should give hints on how to fill the input.

Mandatory field

Enter your name

Disabled field

Enter your name 
<!-- SIMPLE INPUT -->
  <c3m-input
    [id]="'name1'"
    name="name1"
    placeholder="a placeholder">
    Enter your name
  </c3m-input>

<!-- MANDATORY INPUT -->
  <c3m-input
    [id]="'name2'"
    name="name2"
    required="required">
    Enter your name
  </c3m-input>

<!-- DISABLED INPUT -->
  <c3m-input
    [id]="'name3'"
    name="name3"
    value="Peter Parker"
    disabled="disabled">
    Enter your name
  </c3m-input>

Search Input

Search the site

The input type search will inherit all attributes from the generic input element.

<!-- INPUT SEARCH -->
  <c3m-input-search
    [id]="'main-search'"
    name="main-search">
    Search the site
  </c3m-input-search>

Phone Input

Enter your phone number

The input type tel will inherit all attributes from the generic input element. In addition, you may use also:

pattern
regular expression against which the control's value is to be checked (optional)
title
gives description of the pattern (mandatory if pattern is used)

If you're using the pattern attribute, you should add a title attribute to give a description of the pattern.

For pattern examples, check HTML Pattern.

  <!-- INPUT PHONE -->
  <c3m-input-tel
    [id]="'phoneTelId'"
    name="phoneTelName"
    placeholder="+xx x.xx.xx.xx.xx">
    Enter your phone number
  </c3m-input-tel>

Password Input

Password

The input type password will inherit all attributes from the generic input element.

<c3m-input-password
  id="'myPwd'"
  name="myPwd">
  required="required">
  Password
</c3m-input-password>

E-mail Input

Enter your e-mail adress

The input type email will inherit all attributes from the generic input element. In addition, you may use also:

multiple
If present, allows the user to enter more than one value (optional)
  <!-- INPUT MAIL -->
    <c3m-input-mail
      [id]="'mailId'"
      name="mailName"
      placeholder="peter-parker@dailybugle.com">
      Enter your e-mail adress
  </c3m-input-mail>
