Tooltip Slider

The current value is displayed in a sliding tooltip.

Slider with Bubble

Just add the class bubble when calling the c3m-slider component to display the output element as a tooltip.

<!-- SIMPLE SLIDER -->
  <c3m-slider
    [id]="'slider1'"
    name="checkboxName1"
    min="0"
    max="50"
    value="45"
    step="1">
    Choose a value
</c3m-slider>

Progress Bar

Progress
id
single field identifier
name
general field identifier
value
indicates progress (floating between 0 and max)
max
maximum progress value, defaults to 1 (optional)
disabled
when present, the element is disabled/read-only (optional)
<!-- SIMPLE PROGRESS BAR -->
  <c3m-progress
    [id]="'prog1'"
    name="prog1"
    title="Progress"
    max="100"
    value="30">
    Progress
  </c3m-progress>
